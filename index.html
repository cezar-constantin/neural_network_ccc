Stage 1

From sketch to tensor

The handwritten canvas is compressed into a centered 28x28 grayscale signal, then flattened into 784 inputs.

Stage 2

Hidden layers form structure

Early neurons respond to local strokes, while the second hidden layer combines them into digit-like parts.

Stage 3

Output probabilities settle the guess

The ten output neurons compete through softmax, producing a probability distribution over digits 0 through 9.

Activation map

Neuron activation

Zoom 100%

Use the zoom controls to inspect dense regions of the diagram. On smaller screens, swipe horizontally inside the network frame to explore the full path from input pixels to final probabilities.

Input layer

Every bright pixel becomes an activation.

After normalization, the sketch is flattened into 784 scalar values that seed the rest of the network.

Hidden layer 1

Stroke detectors come online first.

These neurons respond to local diagonals, terminals, bends, corners, and simple stroke pressure patterns.

Hidden layer 2

Those strokes combine into digit parts.

Curves, loops, bars, and partial circles emerge here before the output layer turns them into a class decision.

Electric blue

Positive weights increase the next neuron's response when the source neuron is active.

Amber contrast

Negative weights pull the next neuron down and weaken that connection path.

Opacity and width

Brighter, thicker edges indicate stronger active influence through the current prediction pass.

Output layer

Digit probabilities

This probability panel is shown separately from the network diagram so the final class scores can be read clearly without sitting on top of the output-neuron view.

Feature maps

How the learned filters are visualized

Each thumbnail below represents a learned weight pattern. Brighter cyan and blue areas correspond to positive weights, while warm contrast tones indicate negative influence. Dark regions are near zero and contribute only weakly.

Layer 1 uses cyan to emphasize early stroke detectors.
Layer 2 uses blue to show richer composite patterns.
Warm tones mark negative weights that suppress activation.
Deep navy indicates values close to zero.
